## v1.0.0

### Change Description

**New Features<a name="section11862975"></a>**

Provides non-equivalence optimization patch and equivalence optimization patch. The non-equivalence optimization patch further optimizes the IVFPQ algorithm and supports the HNSW FP16 interface. The patch is released on Gitcode, and the code branch version is `v1.0.0`.

**Modified Features<a name="section16450949161512"></a>**

None

**Deleted Features<a name="section9218125814159"></a>**

None

## V25.3.0

### Change Description

**New Features<a name="section11862975"></a>**

The Faiss library is added, and the code is open-sourced. This library can be used only after being compiled using the source code. The patch is released on Gitcode, and the code branch version is `v1.8.0-2512`.

**Modified Features<a name="section16450949161512"></a>**
